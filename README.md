# DS-GA-1012-Natural-Language-Understanding

- **HW1:** Preprocessed the **SMS Spam Collection Dataset** and trained a **bag-of-words logistic regression classifier** for spam detection.  
- **HW2:** Reused the **spam prediction dataset** from HW1 and implemented a **word-level BiLSTM sentence encoder** with a neural network classifier.  
- **HW3:** Performed **Bayesian hyperparameter optimization** on a **RoBERTa model** fine-tuned for the **BoolQ (Boolean Question Answering)** dataset.
